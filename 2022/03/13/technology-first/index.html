一些会用到的步骤

创建文档：右键wakingdead.github.io文件夹，点击GIT Bash Here,输入代码hexo new post xxx。在Front-matter栏目中填写update、date、tags、categories。

更改主题：需要在_config.yun.yml文件中，按照自己的需要更改主题。更改完成后，右键wakingdead.github.io文件夹，点击GIT Bash Here,输入代码npm i hexo-theme-yun@latest。

写新的文档：右键wakingdead.github.io文件夹，点击GIT Bash Here,输入代码hexo new post xxx,即可新建xxx.md文件，在文档中内容即可

更新旧的文档：在旧的文档的Front-matter栏目，添加update:20xx-xx-xx，然后修改/更新即可

更新静态文件：完成更新以后，右键wakingdead.github.io文件夹，点击GIT Bash Here,输入代码hexo clean删除原来的静态文件，然后输入代码hexo generate生成新的静态文件

部署网站：hexo deploy

备份代码到GitHub

git add -A
git commit -m "这次做了什么更改，简单描述下即可"
git push
